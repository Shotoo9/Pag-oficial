Tipos de redes

Tipos de Redes: ¿Cómo están conformadas?

Las redes se clasifican principalmente por su alcance geográfico y uso:

Red LAN (Local Area Network)

Red de área local. Cubre espacios pequeños como una casa, oficina o escuela.

Se conforman asi:

Dispositivos (PCs, impresoras, smartphones).

Switch o router.

Cables Ethernet o conexión Wi-Fi.

Dirección IP interna.

Ejemplo de uso:

Red de computadoras en un colegio para compartir archivos e impresoras. Red doméstica para acceder a internet desde varios dispositivos.

Red MAN (Metropolitan Area Network)

Red de área metropolitana. Cubre una ciudad o campus universitario.

Se conforma de:

Varias LAN interconectadas.

Conexión por fibra óptica o enlaces inalámbricos de alta velocidad.

Gestión centralizada.

Ejemplo de uso:

Red que conecta las diferentes sedes de una universidad en una ciudad. Servicio de Wi-Fi municipal gratuito en una ciudad.

Red WAN (Wide Area Network)

Red de área amplia. Cubre grandes distancias geográficas, incluso países y continentes.

Se conforma de:

Varias LAN y MAN interconectadas.

Infraestructura pública y privada (proveedores de internet, satélites, cables submarinos).

Protocolos como MPLS, VPN.

Ejemplo de uso:

Internet (la red WAN más grande). Empresas multinacionales conectando sus oficinas de distintos países.

Red WLAN (Wireless LAN)

Variante de la LAN, pero sin cables.

Se conforma de:

Router Wi-Fi.

Dispositivos inalámbricos.

Seguridad por claves y protocolos (WPA2/WPA3).

Ejemplo de uso:

Wi-Fi en una casa o cafetería. Escuelas con acceso inalámbrico en cada aula.

Red PAN (Personal Area Network)

Red personal, a corta distancia.

Se conforma de:

Bluetooth, NFC, USB.

Smartphones, relojes inteligentes, auriculares.

Ejemplo de uso:

Transferencia de archivos entre celular y notebook por Bluetooth. Conexión de smartwatch con un celular.
